# Code Kata International

Code Kata Bootstrap repository:

## Getting Started

### JavaScript

#### Install test dependencies

Start your project in the `javascript` directory:

```
$ cd javascript
$ npm install
```

#### Run the tests:

```
$ npm test
```

Run the tests and watch file changes:

```
$ npm test-watch
```

### PHP

```
$ cd php
$ composer install
```

## Remember to DELETE YOUR CODE!

```
$ git reset --hard
$ git clean -f
```
